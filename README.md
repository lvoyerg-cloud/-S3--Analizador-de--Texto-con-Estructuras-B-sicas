# -S3--Analizador-de--Texto-con-Estructuras-B-sicas
Guarde el archivo como analizador_texto.py. En la terminal ubíquese en la carpeta del archivo y ejecute: python analizador_texto.py. Aparecerá el menú para leer, crear o analizar textos. El sistema ejecuta pruebas automáticas al iniciar. Para prueba manual, elija opción 2, escriba: hola mundo, y consulte hola para verificar frecuencia 2 correcta
